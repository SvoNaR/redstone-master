# Redstone Master

Клиентский Fabric-мод для Minecraft **1.21.11**, обучающий редстоун-механикам.

## Возможности

- Открытие интерфейса по клавише **]** (правая квадратная скобка)
- Окно по центру экрана занимает **80%** площади; по краям виден мир или фон главного меню
- Вкладки: **Обучение**, **Об авторе**, **О моде** (пока заглушка)
- Приветственный экран с текстом о проекте

## Сборка

Требуется **JDK 21**.

```bash
./gradlew build
```

Готовый JAR: `build/libs/redstone_master-1.0.0.jar`

## Запуск в среде разработки

```bash
./gradlew runClient
```

## Управление

| Действие | Клавиша |
|----------|---------|
| Открыть / закрыть окно мода | `]` |
| Закрыть окно | Кнопка **X** или `Esc` |

Клавишу можно переназначить в **Настройки → Управление → Redstone Master**.

## Структура

- `src/client/java/ru/redstonemaster/` — клиентский код (GUI, клавиши)
- `src/main/resources/` — `fabric.mod.json`, переводы, ресурсы
- `Исходники/` — макеты интерфейса
